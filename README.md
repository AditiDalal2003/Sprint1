# Air Quality Prediction System

## Project Description
This project predicts Air Quality Index (AQI) using machine learning models.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Flask/streamlit
- Jupyter Notebook

## Project Structure
- data/ : Dataset
- notebooks/ : Analysis
- src/ : Source code
- models/ : Trained models
- app/ : Flask/streamlit app

## How to Run
1. Install requirements
2. Train model
3. Run Flask/streamlit app
